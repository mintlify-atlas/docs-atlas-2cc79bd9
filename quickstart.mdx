## Get Started

This guide will help you set up and run Kids&Us App Versions locally in just a few minutes.

<Steps>
  <Step title="Prerequisites">
    Ensure you have the following installed:
    - **Node.js** version 18.17.0 or higher
    - **npm** (comes with Node.js)

    Check your versions:
    <CodeGroup>
    ```bash npm
    node --version
    npm --version
    ```
    ```bash yarn
    node --version
    yarn --version
    ```
    ```bash pnpm
    node --version
    pnpm --version
    ```
    </CodeGroup>
  </Step>

  <Step title="Clone the Repository">
    Clone the repository from GitHub:
    ```bash
    git clone https://github.com/marcundertest/kidsandus-app-versions.git
    cd kidsandus-app-versions
    ```
  </Step>

  <Step title="Install Dependencies">
    Install all required packages:
    <CodeGroup>
    ```bash npm
    npm install
    ```
    ```bash yarn
    yarn install
    ```
    ```bash pnpm
    pnpm install
    ```
    </CodeGroup>

    <Note>
    The installation includes Next.js 16, React 19, TypeScript, Tailwind CSS, and all necessary dependencies.
    </Note>
  </Step>

  <Step title="Start Development Server">
    Run the development server:
    <CodeGroup>
    ```bash npm
    npm run dev
    ```
    ```bash yarn
    yarn dev
    ```
    ```bash pnpm
    pnpm dev
    ```
    </CodeGroup>

    The application will start at `http://localhost:3000`
  </Step>

  <Step title="View the Dashboard">
    Open your browser and navigate to:
    ```
    http://localhost:3000
    ```

    You should see the Kids&Us App Versions dashboard displaying version information for all tracked applications.
  </Step>
</Steps>

## Using the Dashboard

### View App Versions

The main table displays:
- **Application name and icon** - Each Kids&Us app
- **Store** - Which app store (Google Play, App Store iOS/macOS, Microsoft Store, Huawei AppGallery)
- **Version** - Current version number (click to copy)
- **Last Update** - When the app was last updated

### Refresh Data

Click the **Update Data** button in the header to fetch the latest version information from all app stores.

<Warning>
Data updates are rate-limited to once every 60 minutes to prevent API abuse. If you try to update too frequently, you'll see a countdown timer.
</Warning>

### Admin Bypass (Optional)

For testing or urgent updates, you can bypass the rate limit:

1. Press the `.` (dot) key anywhere on the page
2. Enter the admin key: `force`
3. The data will refresh immediately

<Note>
See [Admin Force Update](/admin/force-update) for complete documentation on the admin bypass feature.
</Note>

### Switch Themes

Click the sun/moon icon in the header to toggle between light and dark modes. Your preference is saved automatically.

## API Access

You can also access the version data programmatically via the REST API:

```bash
# Get current app version data
curl http://localhost:3000/api/data

# Trigger a data refresh
curl -X POST http://localhost:3000/api/update
```

See the [API Documentation](/api/data) for complete endpoint details.
